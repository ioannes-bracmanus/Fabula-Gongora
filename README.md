Edición de la obra "Fábula de Polifemo y Galatea" de Luis de Góngora, con bibliografía crítica. El sistema de normas o estructura bibliográfica usada es [NNCPh](https://github.com/juan-bracled/NNCPh-Novae-normae-ad-citationem-philologicam.git), que se puede consultar, descargar, usar y modificar. El mini-paquete editio-critica.sty sirve para limpiar el preámbulo de parte del código para comprimirlo en un paquete local; con el archivo .sty dentro de la carpeta en la que descomprima el documento, podrá compilar.
