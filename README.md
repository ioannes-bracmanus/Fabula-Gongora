Edición de la obra "Fábula de Polifemo y Galatea" de Luis de Góngora, con bibliografía crítica. El sistema de normas o estructura bibliográfica usada es [ASCPh](https://github.com/juan-bracled/NNCPh-Novae-normae-ad-citationem-philologicam.git), que se puede consultar, descargar, usar y modificar. El mini paquete local editio-critica.sty sirve para limpiar el preámbulo de parte del código para comprimirlo; con el archivo .sty ubicado en la carpeta descomprimida podrá compilar el documento.
